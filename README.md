### 定时执行

特别注意:电信官方服务器经常抽风,导致不成功所以让他每八小时执行一下,一天三次.显示空白是服务器抽风,显示聚合任务接口成功才是请求成功了

Linux 可使用 Crontab 设置定时任务

windows可使用 系统任务计划程序 设置定时任务
